Overview

Create annotator.js annotations that are anchored to portions of images.

Try it out in this simple image annotation demo.

Installation

Using this annotator plugin requires imgareaselect CSS and javascript and the annotator-imgselect javascript. A minified version of the combined imgareaselect and annotator-imgselect are provided for convenience.

Include the appropriate CSS and Javascript files:

{% highlight html %} {% endhighlight %}

In your annotator application initialization, load the module and configure it with a selector for images that can be annotated.

{% highlight html %} var app = new annotator.App(); app.include(annotatorImageSelect, { element: $('.content img'), }); {% endhighlight %}
